# SeatTooLong — 产品需求文档 (PRD)

## 1. 产品概述

| 项目 | 内容 |
| --- | --- |
| **产品名称** | SeatTooLong |
| **平台** | Windows 10/11 桌面应用 |
| **技术栈** | C# + WPF (.NET 8) |
| **一句话描述** | 通过 USB 摄像头实时检测用户是否在座，连续久坐超过阈值后弹窗+悬浮窗提醒用户起身活动 |

## 2. 目标用户与问题

- **目标用户**：长时间伏案工作的办公室人群
- **核心问题**：久坐危害健康，用户沉浸工作时往往忘记起身
- **解决方案**：摄像头被动检测 + 智能提醒，无需手动打卡

## 3. 功能需求

### 3.1 核心功能 — 久坐检测与提醒

| 参数 | 默认值 | 可配置 |
| --- | --- | --- |
| 久坐阈值 | **45 分钟** | ✅ 用户可调 |
| 建议休息时长 | **5 分钟** | ✅ 用户可调 |
| 检测间隔 | 每 2 秒采样一帧 | ✅ |
| 离开宽限期 | **2 秒** | ✅ 用户可调 |

**检测逻辑**：

1. 定时从摄像头采集帧画面，默认仅在内存中分析
2. 优先使用 OpenCV DNN + UltraFace ONNX 模型检测正脸，并在均衡化/轻微旋转后的图像上重试
3. 正脸未命中时，回退到侧脸与上半身级联规则，提高桌前姿态识别率
4. 连续检测到"在座"累计达到久坐阈值 → 触发提醒
5. 连续无人脸/人体且超过离开宽限期 → 结束当前久坐并进入空闲/休息流程
6. 休息倒计时结束（或检测到用户回来）→ 进入下一轮计时

**边界处理**：

- 短暂遮挡或暂时离席（低于离开宽限期）不视为离开，避免误判
- 休息不足建议时长就回来 → 悬浮窗提示"建议再休息 X 分钟"

### 3.2 提醒方式（两种并行）

#### A. 系统托盘通知弹窗

- Windows 原生 Toast Notification
- 显示："您已连续坐了 45 分钟，建议站起来活动 5 分钟！"
- 可点击"知道了"关闭 或 "稍后提醒"（延迟 5 分钟再提醒）

#### B. 桌面悬浮窗

- **常驻模式**：小巧悬浮窗实时显示"已坐 XX:XX"计时
- **提醒模式**：久坐到阈值后，悬浮窗变色（红色/橙色）+ 显示休息倒计时
- 可拖拽移动位置，可最小化/隐藏
- 置顶显示，不遮挡主要工作区域

### 3.3 摄像头管理

- 启动时自动枚举可用摄像头设备
- 设置页面提供下拉列表手动选择摄像头
- 托盘菜单提供“预览当前摄像头”，便于确认取景、遮挡和设备状态
- 打开摄像头时自动在 DirectShow / Media Foundation / OpenCV 默认后端之间回退；读帧失败后自动重试
- 摄像头不可用时 → 系统托盘图标变灰 + 提示
- **隐私保障**：画面仅在内存中实时分析；只有用户手动开启录制时才会保存本地素材

### 3.4 统计报表（详细）

**数据记录**（存入本地 SQLite）：

- 每次久坐的开始时间、结束时间、持续时长
- 每次休息的开始时间、结束时间、持续时长
- 每日汇总：总久坐时长、起身次数、最长连续久坐

**报表界面**：

- **今日概览**：当日久坐总时长、起身次数、最长连续久坐、久坐/休息时间轴
- **历史趋势**：折线图展示近 7 天 / 30 天的久坐时长变化
- **柱状图**：每日起身次数对比
- 支持按日期范围筛选

### 3.5 系统托盘与开机自启

- 最小化到系统托盘运行，右键菜单：
  - 打开主界面
  - 暂停/恢复监测
  - 开始/停止录制检测优化素材
  - 预览当前摄像头
  - 查看今日统计
  - 设置
  - 退出
- 支持注册 Windows 开机自启动
- 启动后自动最小化到托盘

### 3.6 多语言支持

- 支持**中文**和**英文**双语界面
- 设置中可切换语言，即时生效
- 使用 `SeatTooLong.Core.Localization.LocalizationService` 集中管理 UI 文案

### 3.7 检测优化素材录制（可选）

- 托盘菜单支持手动开始/停止录制，用于回归测试和检测效果调优
- 录制默认关闭，仅在用户主动开启后保存 JPEG 帧与同名 JSON 元数据
- 录制文件保存在 `%LOCALAPPDATA%\SeatTooLong\recordings\<session>`，不会自动上传或同步到外部服务

## 4. 非功能需求

| 维度 | 要求 |
| --- | --- |
| **性能** | CPU 占用 < 5%（常规办公设备，采样间隔 2–5 秒），内存 < 100MB |
| **隐私** | 零数据外传，零图片存储，纯本地运行 |
| **兼容性** | Windows 10 1903+ / Windows 11 |
| **安装** | 提供 self-contained x64 安装包（Inno Setup），支持简体中文安装界面 |
| **稳定性** | 7×24 小时后台运行不崩溃，摄像头断开自动恢复 |

## 5. 技术架构概要

```text
┌─────────────────────────────────────┐
│            WPF UI Layer             │
│  ┌──────────┐  ┌─────────────────┐  │
│  │ 悬浮窗   │  │ 主窗口(设置/报表/ │  │
│  │ /预览窗  │  │ 摄像头预览)      │  │
│  └──────────┘  └─────────────────┘  │
├─────────────────────────────────────┤
│          Business Logic             │
│  ┌──────────┐  ┌──────────────┐     │
│  │ 状态机   │  │ 通知管理器   │     │
│  │ (计时/   │  │ (Toast/悬浮窗)│     │
│  │  阈值)   │  └──────────────┘     │
│  └──────────┘  ┌──────────────┐     │
│                │ 统计服务     │     │
│                │ (SQLite)     │     │
│                └──────────────┘     │
├─────────────────────────────────────┤
│         Camera & Detection          │
│  ┌──────────────┐ ┌──────────────┐  │
│  │ OpenCvSharp  │ │ 人脸/人体    │  │
│  │ (摄像头采集) │ │ 检测器       │  │
│  └──────────────┘ └──────────────┘  │
└─────────────────────────────────────┘
```

**关键技术选型**：

- **摄像头采集**：OpenCvSharp4（OpenCV 的 C# 绑定）
- **桌前在座检测**：OpenCV DNN + UltraFace ONNX 正脸检测，结合侧脸/上半身 Cascade 兜底
- **数据库**：SQLite（via Microsoft.Data.Sqlite）
- **图表**：LiveCharts2 或 OxyPlot
- **通知**：Microsoft.Toolkit.Uwp.Notifications（Toast）
- **开机自启**：注册表 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`
- **本地化与安装**：Core LocalizationService + PowerShell/Inno Setup 打包链路

## 6. 状态机设计

```text
         ┌──────────┐
    ┌───▶│  空闲    │◀──── 检测到离开
    │    │(无人在座)│
    │    └────┬─────┘
    │         │ 检测到人
    │    ┌────▼─────┐
    │    │  久坐中  │──── 悬浮窗显示已坐时长
    │    │(计时累加)│
    │    └────┬─────┘
    │         │ 达到阈值
    │    ┌────▼─────┐
    │    │  提醒中  │──── 弹通知 + 悬浮窗变色
    │    │(等待离开)│
    │    └────┬─────┘
    │         │ 检测到离开
    │    ┌────▼─────┐
    │    │  休息中  │──── 悬浮窗显示休息倒计时
    │    │(倒计时) │
    │    └────┬─────┘
    │         │ 倒计时结束/回来太早则提示
    └─────────┘
```

## 7. 设置项汇总

| 设置项 | 类型 | 默认值 |
| --- | --- | --- |
| 久坐阈值 | 滑块/输入框（分钟） | 45 |
| 建议休息时长 | 滑块/输入框（分钟） | 5 |
| 检测采样间隔 | 下拉（秒） | 2 |
| 离开宽限期 | 滑块（秒） | 2 |
| 摄像头选择 | 下拉列表 | 默认摄像头 |
| 开机自启 | 开关 | 开 |
| 语言 | 下拉（中文/English） | 跟随系统 |
| 悬浮窗显示 | 开关 | 开 |
| 悬浮窗透明度 | 滑块 | 80% |

## 8. MVP 分期建议

| 阶段 | 内容 |
| --- | --- |
| **P0 — MVP** | 摄像头检测 + 久坐计时 + Toast 通知 + 托盘常驻 |
| **P1** | 悬浮窗（实时计时 + 休息倒计时） + 设置界面 |
| **P2** | SQLite 统计 + 报表界面（今日概览 + 历史图表） |
| **P3** | 多语言 + 开机自启 + 安装包打包 |
